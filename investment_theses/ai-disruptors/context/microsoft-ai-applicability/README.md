# Microsoft AI Applicability Framework
## Source: "Working with AI: Measuring the Applicability of Generative AI to Occupations"
- **Authors:** Kiran Tomlinson, Sonia Jaffe, Will Wang, Scott Counts, Siddharth Suri (Microsoft Research)
- **Paper:** arXiv:2507.07935 | **Data:** github.com/microsoft/working-with-ai
- **Dataset:** 200k anonymized Bing Copilot conversations, Jan-Sep 2024, US users
- **Coverage:** 785 SOC occupation codes, 149.8M US workers

## Purpose of This Extract
Condensed reference for identifying companies whose products/services target industries
with high AI applicability — i.e., industries ripe for AI-driven disruption or augmentation.

## Files in This Directory
- `scoring_methodology.md` — How AI applicability scores are computed
- `occupation_rankings.md` — Which occupations score highest/lowest and why
- `disruption_framework.md` — User-goal vs AI-action patterns, delegation vs collaboration
- `investment_signals.md` — Derived signals for identifying disruptive companies
